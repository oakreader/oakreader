最近做了一款 Mac 原生的 AI 阅读工具 OakReader。读 PDF 论文/电子书，浏览网页的时候，遇到困惑，一个按钮打开侧边栏，圈中一段文字就能和你喜欢的 AI 模型讨论问题，再也不用切到 ChatGPT / Claude / DeepSeek 去聊天。

阅读过程中的思考，一个按钮打开笔记插件，能像 Flomo / Slack / 微信 一样随手记笔记。阅读外文文献想要翻译的时候，一键打开内置翻译插件。还内置了浏览器功能，可以直接浏览收藏网页。

对于文献管理，这款工具也支持了 Zotero 同样的功能，包括目录标签管理和引用管理。对于 AI 模型配置，可以配置你的 API Key，或者是通过你ChatGPT 账号 OAuth 登录，使用完全免费。
