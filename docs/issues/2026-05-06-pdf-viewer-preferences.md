# PDF Viewer Preferences

## Summary

Add display options and per-document reading state to the PDF viewer.

## Requirements

- [ ] Double-page spread mode
- [ ] Remember zoom and scroll position per document
- [ ] Scroll mode: continuous vs. paginated
- [ ] Page gap between pages in continuous mode

## Affected Areas

- `Preferences.swift` — viewer keys
- PDF viewer coordinator — per-document state persistence
