# Global Font Settings

## Summary

Add a global font preference (family and size) applied across the app UI.

## Requirements

- [ ] Global font family picker (system / serif / sans-serif)
- [ ] Base font size (numeric, e.g. 13-18pt, default 14pt)
- [ ] Notes editor inherits unless overridden

## Affected Areas

- `Preferences.swift` — `globalFontFamily`, `globalFontSize`
- Settings UI — section in General settings
